This plugin generate shortcode for Axeptio consents displaying


== Description ==

This plugin get all consents available from an Axeptio account and generate shortcodes for each one.

Features include :

* Display all consents available in back-office
* Display shortcode for each consent
* Required parameter to shortcode if consent is in a form
* Contact Form 7 compatibility

== Installation ==
1. In your WordPress admin panel, go to "Plugins > New Plugin", search for "Axeptio" and click "Install now"
1. Upload the entire "axeptio" folder to the /wp-content/plugins/ directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
